Mes projets

Portfolio Personnel

Projet solo de 6 semaines visant à créer un portfolio personnel pour présenter mes compétences et projets en développement web.

Développement d'un site web interactif et responsive servant de vitrine pour présenter mes projets et compétences. Construit en HTML5, CSS3 et JavaScript, il intègre un système de thèmes (mode clair/sombre), un switch de langue (FR/EN) avec persistance via localStorage, et une mise en page structurée en sections. Les projets sont affichés sous forme de cards interactives avec effets de hover, et des liens vers les github des projets, ainsi qu'un bouton permettant de télécharger mon Curriculum Vitae au format PDF.

Athlia

Projet de fin d'année en groupe de 3 personnes sur 1 mois et demi pour créer une application fitness interface 3D interactive.

Le front-end est développé avec React.js, CSS pour l'interface et Three.js pour la modélisation 3D des exercices et parcours. Le back-end repose sur Django avec une architecture en trois couches (Présentation, Business Logic, Persistance) et l'utilisation du Facade Pattern pour simplifier les interactions. La base de données est gérée avec PostgreSQL, assurant la gestion des profils utilisateurs et le suivi des performances (sessions, progression). L'application intègre des données dynamiques en temps réel via WebSockets, et une documentation technique détaillée incluant mockups, user stories, API REST et diagrammes UML.

HBnB (Airbnb Clone)

Projet de groupe de 3 personnes sur 6 semaines pour créer un clone d'Airbnb avec interface web et API RESTful.

Le back-end est développé en Python avec le framework Flask, utilisant SQLAlchemy pour la gestion de la base de données relationnelle (MySQL). L'API RESTful permet la gestion des utilisateurs, logements, réservations et avis. Le front-end est développé avec HTML5 et CSS3 pour le front-end et JavaScript pour les interactions dynamiques. Le projet suit une architecture modulaire, utilise le DOM manipulation pour les mises à jour en temps réel, et intègre des fonctionnalités avancées comme des formulaires interactifs et des modales de détail.

Générateur de Compliments

Projet solo de 1 semaines pour créer une application web simple et interactive.

Développé avec HTML5 et CSS3 pour la structure et le style, et JavaScript pour la logique de génération aléatoire des compliments, stockés directement dans quelques lignes de code. L'interface inclut un background animé et une musique de fond pour améliorer l'expérience utilisateur. Le projet est interactif, avec un bouton permettant de générer de nouveaux compliments et un affichage dynamique sur la page.

Shell en C

Projet en duo de 2 semaines pour créer un mini-shell en C.

Développement d'un interpréteur de commandes (mini-shell) en langage C capable d'exécuter des commandes système de façon interactive. Gestion des commandes internes (cd, exit) et externes, redirections (>, <) et pipes (|). Création de processus fils via fork() et exécution de commandes avec execve(). Gestion des signaux (Ctrl+C) et optimisation de la mémoire.
